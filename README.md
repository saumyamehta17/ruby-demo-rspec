## How to suppress Cached Value in TestCases(Rspec)

### Setup Ruby Project

create a folder

```
mkdir ruby-demo-rspec
cd ruby-demo-rspec
```

create a Gemfile

```
touch Gemfile
```

then

```
bundle install
```

add Rspec

```
rspec --init
```

create a spec file for this demo

```
touch test_spec.rb
```

To run test cases

```
rspec 
```
